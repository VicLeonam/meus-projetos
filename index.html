Tags principais que formam a estruta do site

No HTML, a tag <html>; é usada para definir o início e o fim do documento HTML, indicando ao navegador que o conteúdo está em formato HTML. A tag <head>; é usada para conter informações sobre o documento, como o título da página e metadados. Por fim, a tag <body>; é usada para conter o conteúdo visível da página, como texto, imagens e outros elementos que serão exibidos no navegador.

Hierarquia de titulos

A hierarquia de títulos no HTML é como um sistema de títulos e subtítulos em um texto. O <h1>; é o título principal, usado apenas uma vez por página, e o <h2> é um subtítulo importante que pode ser usado várias vezes. O <h3>, <h4>, <h5>, e <h6> são usados para subtítulos menos importantes, seguindo a mesma lógica.

Background Color (Cor de Fundo)

Em HTML, você pode definir a cor de fundo de uma página ou de um elemento usando o atributo bgcolor dentro da tag <body>. Por exemplo, <body> bgcolor="blue"> define o fundo da página como azul. No entanto, o uso de estilos em cascata (CSS) é mais comum e recomendado para controlar a aparência visual, incluindo cores de fundo, em páginas web modernas.

Tags de parágrafos e quebra de linha

Em HTML, a tag <p> é usada para definir parágrafos de texto. Por exemplo, <p>Este é um parágrafo.</p> cria um parágrafo simples.
Já a quebra de linha é feita com a tag <br>, que indica ao navegador para iniciar uma nova linha. Por exemplo, <br>cria uma quebra de linha no texto.

Tags de Formatação de Texto

Em HTML, as tags de formatação são usadas para alterar a aparência do texto em uma página web. Alguns exemplos comuns são:

  • <strong>: Deixa o texto em negrito.
  • <em>: Enfatiza o texto, geralmente exibindo em itálico.
  • <u>: Sublinha o texto.
  • <s>: Aplica uma linha de texto riscado, indicando deleção.
  • <sup>: Eleva o texto, como em um expoente.
  • <sub>: Abaixa o texto, como em um índice.
Essas tags podem ser usadas individualmente ou combinadas para obter diferentes efeitos de formatação no texto.

Alinhamento de Texto

Em HTML, você pode alinhar o texto horizontalmente dentro de um parágrafo usando o atributo <align>. Por exemplo, <p> align="center"> centraliza o texto dentro do parágrafo, <p> align="left"> alinha o texto à esquerda e <p> align="right"> alinha o texto à direita. No entanto, o uso de CSS é mais recomendado para estilos de alinhamento mais avançados e para separar o estilo do conteúdo.

Tag de imagem

A inserção de imagens é possível com a tag <img>. Para inserir uma imagem em uma página é preciso usar a propriedade src que deverá apontar para o local e o nome do arquivo da imagem a ser utilizada.

OBS:A imagem precisa estar salva dentro da pasta que esta seu codigo html

Vinculação de links em imagens


Em Html, você pode vincular um link a uma imagem usando a tag <a> em torno da tag <img>. Isso vai permitir você clicar na imagem e ser redirecionado para outro site

Listas em Html

Em HTML, você pode criar listas ordenadas <ol> (ordered list) e não ordenadas <ul> (unordered list). Em seguida, dentro dessas tags, você pode usar a tag <li>(list item) para cada item da lista. Por exemplo:
